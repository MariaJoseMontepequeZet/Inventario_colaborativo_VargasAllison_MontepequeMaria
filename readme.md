# Introducción 📄
Aplicación de consola para la gestión técnica de activos. El sistema centraliza el control de existencias y la valoración financiera, garantizando una precisión de 4 decimales en todos los cálculos contables.

---

### ¿Qué hace este proyecto exactamente? 🤔

* Administración de Inventario: Registro, actualización y depuración de existencias.

* Reportabilidad: Consulta detallada de productos en tiempo real.

* Cálculo Financiero: Valoración exacta del stock total mediante la operación:

---

### Antes de empezar 🫸🏼
Para ejecutar el proyecto, asegúrate de contar con los siguientes elementos:
* **Python**: Intérprete instalado (versión 3.x recomendada).
* **Terminal**: Acceso a una terminal de comandos (PowerShell, Bash o Zsh).
* **Git**: Herramienta instalada para la gestión de versiones.

---

### Cómo ponerlo a andar (Paso a paso) 🚶🏽‍♂️‍➡️

1. **Descarga el código:**
    Clona el repositorio desde GitHub utilizando el siguiente comando en tu terminal:
